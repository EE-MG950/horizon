# 🌍 Earth Visibility 3D & Analytics

Интерактивный веб-инструмент для визуализации и расчета площади видимой поверхности Земли в зависимости от высоты наблюдателя. Проект сочетает в себе 3D-рендеринг глобуса и динамическую инфографику.



## 🚀 Основные возможности

* **3D Визуализация**: Использование `WebGL Earth 2.0` для отображения зон видимости непосредственно на сферической модели планеты.
* **Динамические расчеты**: 
    * Расчет площади сферического сегмента ($S = 2\pi R^2 \frac{h}{R+h}$).
    * Расчет расстояния по поверхности (длина дуги) до горизонта.
* **Многослойность**: Поддержка нескольких независимых зон видимости одновременно.
* **Кастомизация**: Настройка высоты (от 0.1 м до 100 000+ км), координат, цвета и прозрачности каждой зоны.
* **Смена картографии**: Переключение между детализированными спутниковыми снимками (Esri) и схематичной картой (CartoDB).
* **Аналитика**: Встроенные графики на `Chart.js`, демонстрирующие:
    1.  Процент видимой поверхности на космических высотах (до 40 000 км).
    2.  "Микро-эффект" расширения горизонта на первых 100 сантиметрах подъема.

## 🛠 Технологический стек

* **Frontend**: HTML5, CSS3 (Modern UI, Glassmorphism).
* **3D Engine**: [WebGL Earth API](http://www.webglearth.com/) (на базе Cesium).
* **Charts**: [Chart.js](https://www.chartjs.org/).
* **Math**: Сферическая тригонометрия для расчета координат полигонов и геодезических расстояний.

## 📐 Математическая справка

Проект решает проблему визуального "выворачивания" полигонов на больших высотах. 
Для стабильной отрисовки используется ограничение центрального угла $\theta$:
$$\theta = \arccos\left(\frac{R}{R + h}\right)$$
Где $R$ — радиус Земли (6,371 км), а $h$ — высота наблюдателя. 

## 💡 Как пользоваться

1.  Нажмите кнопку **"+ Добавить область"**.
2.  В панели управления введите нужную высоту (в метрах).
3.  **Кликните на любое место на глобусе**, чтобы мгновенно переместить туда центр выбранной зоны.
4.  Используйте ползунки прозрачности и палитру цветов для визуального сравнения разных высот (например, обзор с самолета против обзора с МКС).
5.  Прокрутите страницу вниз, чтобы увидеть графические зависимости.
